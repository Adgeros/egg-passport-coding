# egg-passport-coding

## Install

```bash
$ yarn add egg-passport-coding
```

## Usage

```js
// {app_root}/config/plugin.js
exports.passportCoding = {
  enable: true,
  package: 'egg-passport-coding',
};
```

## Configuration

```js
// {app_root}/config/config.default.js
exports.passportCoding = {
    key: "",
    secret: ""
};
```
